# 글꼴

## font-style : 글자의 기울기
- normal : 기울기 없음
- italic : 이텔릭체

## font-weight 
- normal, 400
- bold, 700
- 100~900

## font-size
- 16px
- 단위 : px, em, rem 등 단위 지정

## lien-height
- 숫자 : 요소의 글꼴 크기의 배수로 지정
- 단위 : px, em, rem 등의 단위로 지정

## font-family : 글꼴1, "글꼴", ... 글꼴 계열;
- serif
- sans-serif *
- monospace
- cursive
- fantasy
